# Layout Inspiration
Study these two pages for layout patterns before building:
1. https://www.leafdesign.co/gaining-ground
2. https://www.leafdesign.co/work/ssa-architects

Key patterns to extract:
- Small-caps muted labels above headings
- Large editorial headings with generous line height
- Body text constrained, never full width
- Full-bleed images breaking out of content container
- Whitespace as the only section separator
- Flat, no shadows, no decorative elements
